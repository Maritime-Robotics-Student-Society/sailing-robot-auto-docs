sailing_robot.pid_control module¶

A simple PID controller class.

This is a mostly literal C++ -> Python translation of the ROS control_toolbox Pid class: http://ros.org/wiki/control_toolbox.

class sailing_robot.pid_control.PID(p_gain, i_gain, d_gain, i_max, i_min)[source]¶

Bases: object

A basic pid class.

This class implements a generic structure that can be used to create a wide range of pid controllers. It can function independently or be subclassed to provide more specific controls based on a particular control loop.

In particular, this class implements the standard pid equation:

$command = -p_{term} - i_{term} - d_{term} $

where:

$ p_{term} = p_{gain} * p_{error} $ $ i_{term} = i_{gain} * i_{error} $ $ d_{term} = d_{gain} * d_{error} $ $ i_{error} = i_{error} + p_{error} * dt $ $ d_{error} = (p_{error} - p_{error last}) / dt $

given:

$ p_{error} = p_{state} - p_{target} $.
